# 🌸 Happy Birthday Ayis

Halaman ucapan ulang tahun yang dibuat dengan React dan penuh cinta.

## Cara Menjalankan

```bash
npm install
npm start
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Upload ke GitHub & Deploy ke GitHub Pages

```bash
# 1. Install gh-pages
npm install gh-pages --save-dev

# 2. Tambahkan ke package.json — "homepage" dan script "predeploy"/"deploy"
# (lihat instruksi lengkap di bawah)

# 3. Deploy
npm run deploy
```
